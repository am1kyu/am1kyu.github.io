Network Honeypot Implementation

Designed and deployed a low-interaction honeypot to study common attack vectors and methodologies used by malicious actors. The project aimed to capture and analyze unauthorized access attempts to understand current threat landscapes.

// Key Features

  • Emulated common services (SSH, FTP, Telnet) to attract attackers.
  • Logged all interaction data, including IP addresses, commands used, and payloads.
  • Configured firewall rules to safely isolate the honeypot.
  • Analyzed collected data to identify patterns and popular attack tools.

// Technologies Used

Python, Linux (Ubuntu), Docker, Wireshark, Log analysis tools
